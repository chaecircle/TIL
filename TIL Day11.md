### 딕셔너리 자료형

> 딕셔너리 구조

- { Key1: Value1, Key2: Value2, Key3: Value3, ....}

```python
>>> information = {'name': '이준혁', 'job': '배우', 'gender': '남성'}

>>> dic1 = {1: 'BeBe', 2: 'Rexha'}

>>> dic2 = {'S': ['한유진', '한유현', '성현제']}
```



- 딕셔너리 쌍 추가하기

```python
>>> a = {1: 'one'}
>>> a[2] = 'two'
>>> a
{1: 'one', 2: 'two'}

>>> a['name'] = '강동원'
>>> a
{1: 'one', 2: 'two', 'name': '강동원'}
```

- 딕셔너리 요소 삭제하기

```python
>>> some = {'이름': '이제훈', '직업': '배우', '성별': '남성'}
>>> del some['이름']
>>> some
{'직업': '배우', '성별': '남성'}
```



#### 딕셔너리 제어

- 딕셔너리에서 Key 사용하여 Value 얻기

```python
>>> grade = {'성현제': 100, '한유진': 77, '한유현': 100}
>>> grade['한유진'] 
>>> 77
>>> grade['성현제']
>>> 100
```
